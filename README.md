# 키워드 배틀
Flask로 만든 키워드 배틀 사이트입니다. 두 개의 키워드를 입력하면 Google에 더 많이 검색된 키워드를 찾아줍니다.

### 설치 및 사용법
```
git clone https://github.com/Mango-Juice/keyword-battle.git
pip install -r requirements.txt
python app.py
```
